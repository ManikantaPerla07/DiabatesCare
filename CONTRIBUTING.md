# Contributing to DiabatesCare

Thank you for your interest in improving DiabatesCare.

## Development Setup

1. Clone the repository.
2. Create and activate a virtual environment.
3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app locally:

```bash
python app.py
```

## Contribution Guidelines

- Keep pull requests focused on one change area.
- Follow existing code style and file structure.
- Prefer clear variable names and small, readable functions.
- Update documentation when behavior or setup changes.
- Test endpoints manually before opening a pull request.

## Pull Request Checklist

- Code runs locally without errors.
- Relevant routes and UI flow are verified.
- README or docs are updated if required.
- Commits are clean and descriptive.

## Reporting Issues

When opening an issue, include:

- Problem summary
- Steps to reproduce
- Expected vs actual behavior
- Environment details (OS, Python version)
- Screenshots or logs if applicable

## Code of Conduct

Be respectful and constructive in all discussions and reviews.
